# 📌 Mô tả bài tập

Bài tập xây dựng một ứng dụng **Barcode Scan App** sử dụng **React Native với Expo **.

Ứng dụng được thiết kế dựa trên giao diện từ **Figma** và sử dụng **React Navigation** để di chuyển giữa các màn hình.

Ứng dụng gồm hai màn hình chính:

* **Home Screen**
* **Scan Screen**

Khi người dùng nhấn **Scan New** trên Home Screen, ứng dụng sẽ **chuyển sang Scan Screen**.
